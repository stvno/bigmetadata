UPDATE {table_name}
  SET {new_numeric_column} =
    OBS_GetMeasure(
      CDB_LatLng(40.7, -73.9),
      'uk.ons.LC2107EW0003'
    );

Measure “Buddhist Population” within an area:

UPDATE {table_name}
  SET {new_numeric_column} =
    OBS_GetMeasure(
      ST_Buffer(CDB_LatLng(40.7, -73.9), 0.01),
      'uk.ons.LC2107EW0003'
    );
